# Species Recognition

## Description
This project is developed using Python and focuses on recognizing different species based on given input data.  
It demonstrates the use of data processing and machine learning concepts in a simple and understandable way.

## Features
- Species classification using Python
- Clean and beginner-friendly code
- Easy to run and understand

## Technologies Used
- Python
- NumPy
- Pandas
- Scikit-learn
